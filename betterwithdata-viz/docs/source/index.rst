
betterwithdata_viz
=====================================

Version: |release|

jupyter notebook widget with a scatter plot and an interactive lens to enable interactive sonification


Quickstart
----------

To get started with betterwithdata_viz, install with pip::

    pip install betterwithdata_viz

or with conda::

    conda install betterwithdata_viz
